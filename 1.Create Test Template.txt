1)	Use live temple from Resharper to create test class template

a)
Go to Resharper >  Tools > Template Explorers > New Template

b)
shortcut Name : testclass

c) Copy and Paste text below


using NUnit.Framework;

namespace $namespace$
{

[TestFixture()]
public class $name$
{

  [SetUp]
  public void Setup()
  {}

  [Test]
  public void $METHOD$()
  {
  //Arrange

  //Act

  //Assert
  }

  [TearDown]
  public void CleanUp()
  {}

}
}

d) Defined $$   Noted: (need to choose All machos)

click on choose macho for 
$namespace$  =>  (i) Default namespace for current file 
                 (ii)untick editable

$name$ => (i)Current filename without extension 
          (ii) untick editable
$Method$ => tick editable
